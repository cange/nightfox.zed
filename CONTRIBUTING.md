# Contributing

Thanks for contributing!

## Guidelines

The following is a set of guidelines for contributing to this project.

- PRs should go to the `main` branch.
- Use [Conventional Commits] rules for writing explicit and meaningful commit messages.

[Conventional Commits]: https://www.conventionalcommits.org/

## Recommendations

- Create a topic branch on your fork for your specific PR.
- If it's your first time contributing to a project then read [About pull requests] on GitHub's docs.
- Consider using `stylua .` to format Lua code to maintain consistent coding styles.

[About pull requests]: https://docs.github.com/en/github/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/about-pull-requests
